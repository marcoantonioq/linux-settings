# linux-settings

Debian + I3wm

# Dependencias

```bash
$ sudo apt install rofi rsync git guake terminator ranger i3blocks yad pavucontrol
```

# Installation

```bash
git clone https://github.com/marcoantonioq/linux-settings.git 
cd linux-settings
bash install.sh
```
